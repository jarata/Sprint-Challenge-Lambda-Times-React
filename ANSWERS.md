1. What are PropTypes used for? Please describe why it's important to type check our data in JavaScript.
PropTypes are used for keeping props bound to their data type. It helps keep errors from occurring by checking each
prop that is passed down is being called correctly.
2. Describe a life-cycle event in React?
The mounting phase is the first of a components life-cycle, this is the start of where the mounted data on state is
before it reaches the updating phase. Here it can be changed, or passed along before heading to the un-mounting phase.
In this last phase the data for the component is de-mounted and rendered. 
3. Explain the details of a Higher Order Component?
A higher order component is function that can receive a component as an argument. These can also be used for storing
other components or data.
4. What are three different ways to style components in React? Explain some of the benefits of each.
Three different style components in React are, CSS, ReactStrap, and Styled-Components. CSS is beneficial because it is
very basic and familiar, Styled-Components is CSS but inside if the JS itself and not separated, and ReactStrap is
great because it has a prebuilt library of styles to use.
